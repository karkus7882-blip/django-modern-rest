## 🚀 How to Run Now

### Quick Start (3 Commands)

```bash
# 1. Navigate to project
cd django-modern-rest/django_test_app

# 2. Start Django server
python manage.py runserver 0.0.0.0:8000

# 3. Open browser
# http://localhost:8000 (same computer)
# http://YOUR_IP:8000 (other devices)
```

### Get Your IP Address

```bash
# Windows
ipconfig

# macOS
hostname -I

# Linux
hostname -I

# Any OS (Python)
python -c "import socket; print(socket.gethostbyname(socket.gethostname()))"
```

## 🔧 Troubleshooting Quick Reference

| Issue | Solution |
|-------|----------|
| Port 8000 busy | Use `python manage.py runserver 0.0.0.0:3000` |
| Can't connect | Make sure you're using `0.0.0.0:8000` not just `runserver` |
| CSS not loading | Clear cache: `Ctrl+Shift+Delete` |
| DisallowedHost error | Add IP to `ALLOWED_HOSTS` in settings.py |
| Images not showing | Check internet (using picsum.photos CDN) |
